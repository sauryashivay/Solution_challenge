Risk Prediction System

AI-powered credit risk analysis using ML + LLM reasoning

📘 About

An intelligent loan risk analysis system that evaluates your financial profile and instantly tells you whether a loan is safe or risky. It not only provides a prediction but also explains the reasoning in simple, human-friendly language so you clearly understand the decision. This helps you make more informed and confident financial choices before borrowing.

✨ Features

    📊 Real-time ML-based risk prediction
    🤖 Explainable AI with LLM-driven reasoning
    ⚖️ Transparency-focused decision insights
    ⚡ Fast and scalable Flask backend API
    🎨 Modern glassmorphism UI design

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Flask (Python)
Machine Learning: Scikit-learn
LLM Integration: Groq / OpenAI APIs
