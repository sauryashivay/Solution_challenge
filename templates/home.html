Risk Prediction System

AI-powered credit risk analysis using ML + LLM reasoning

About

This system predicts whether a loan applicant is a low-risk or high-risk borrower. It combines Machine Learning with Large Language Models to provide both accurate predictions and human-like explanations.

Features

  • 📊 Real-time ML predictions
  • 🤖 AI reasoning using LLM
  • ⚡ Fast API backend
  • 🎨 Glassmorphism UI

Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Flask
ML: Scikit-learn
LLM: Groq / OpenAI APIs
